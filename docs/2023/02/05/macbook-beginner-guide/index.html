Ｍacbook 新手須知

Posted on 2023-02-05 Edited on 2023-02-25

為方便工作效率將經常使用的 Ｍacbook 快速鍵記錄在這裡：

Macbook 快捷鍵一覽

文書處理

刪除檔案

command + delete

顯示隱藏檔案

shift + command + .

剪下檔案並貼上

commmand + C
ctrl + command + V

切換或取消全螢幕

ctrl + command + F

開啟工作管理員

option + command + esc

照片位址

需要特別注意照片匯入時放置在 Finder 圖片資料夾，照片圖庫才能如期查看，若移至照片則會導致圖片位址找不到該檔案

開發相關

快速刪除 terminal 整行指令

ctrl + U

開啟 safari RWD 除錯模式

ctrl + command + R
